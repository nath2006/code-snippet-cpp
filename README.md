# Features

This extension provides snippets for:

- Fast Input Output
- Competitive Programming Template
- Vector
- Pair
- Loop
- Debug Macro
- Typedef
- Sorting
- Binary Search
- Graph
- BFS
- DFS
- Dijkstra
- Segment Tree

---

# How To Use

1. Install the extension
2. Open a `.cpp` file
3. Type the snippet prefix
4. Press:

```txt
Ctrl + Space
```

or press:

```txt
Tab
```

---

# Snippet List

## 1. Basic C++ Template

### Prefix

```txt
cppstartbasic
```

### Function

Create a basic C++ program template.

### Output

```cpp
#include <iostream>
using namespace std;

int main() {

    return 0;
}
```

---

# 2. Fast IO

### Prefix

```txt
cppfastio
```

### Function

Enable fast input output for faster execution when reading large data.

### Output

```cpp
ios::sync_with_stdio(false);
cin.tie(nullptr);
```
